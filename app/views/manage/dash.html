Dashboard

  • Upcoming Events
  • Broadcasts

Volunteer Dashboard - "This is a landing site where volunteers can see the hours they have volunteereds, the events they have attended, the next upcoming event, notifications, and a link to go to their training"

Event Manager Dashboard - "This is a landing site where Event Managers can see an overview of their upcoming event (number of volunteers registered, number of volunteers) and a easy link to send a broadcast for that event. On this dashboard Event Managers can also see other upcoming events, notifications, and link to their other management pages."

Chapter Manager Dashboard - "This is a landing site where Chapter Managers can an overview of their Chapter's next event (number of volunteers registered, number of volunteers) and a easy link to send a broadcast for that event. On this dashboard Chapter Managers can also see other upcoming events, notifications, and link to their other management pages."

Region Manager Dashboard - "This is a landing site where Region Managers can see notifications for upcoming events for the chapters within the region. On this dashboard Region Managers can also see notifications and link to their other management pages."

Admin Dashboard - "This is a landing site where Admins can see an overview of donations in the past month/past year, and the number of upcoming events per region. On this dashboard Region Managers can also see notifications and link to their other management pages."
